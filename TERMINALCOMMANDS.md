Terminal commands

exiftool: tells you a lot of information on the file
echo: for decode and you want it to act like not a command use: echo [word] | base64 –decode
cat: lets you see inside the file
mkdir: makes a new folder
touch: makes a new file
./filename: runs the file
wget: puts the file into your directory that your in
grep -r “anything” : searches all files in directory
find . -name thisfile.txt
cd - : goes back one directory 
Git reflog : looks at modifications made
realpath: tells you where you are 
mv: mv [filename] and then the path to where you want to put it
cd .. : goes back a directory
git revert -m 1 [number] : reverts a commit if you make a mistake. where it says number go to the website and copy the id number and letters
git commit -m "commit message here" : commits changes made to files in a repo
